## Compile

To compile the main program use either:

`make compile`

or

`g++ -std=c++11 bst.cpp main.cpp -o tree

## Run Tests

To run the test use:

`make test`

or

`./test`

## Run Program

To run the program use:

`make run`

or

`./tree`

## Clean

To clean the program use:

`make clean`
